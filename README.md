# Book Tracker – Трекер прочитанных книг

**Автор:** Лапин Илья  


## Описание программы

Book Tracker – это приложение с графическим интерфейсом на Python для ведения списка прочитанных книг.  
Позволяет добавлять книги, фильтровать их по жанру и количеству страниц, сохранять и загружать данные в формате JSON.

## Функциональные возможности

- Добавление книг (название, автор, жанр, количество страниц)
- Валидация полей (не пустые, страницы – число)
- Фильтрация по жанру (частичное совпадение) и по страницам (> заданного числа)
- Удаление выбранной книги
- Сохранение / загрузка в JSON
- Табличное отображение с прокруткой

## Требования

- Python 3.7+
- Стандартные библиотеки: tkinter, json, os

2. Убедитесь, что Python установлен:
   ```bash
   python --version
  
 ## Примеры использования
 1. Заполните поля:
    Название: Война и мир
    Автор: Лев Толстой
    жанр: Роман
    Страниц: 1300
 2. Нажмите кнопку «Добавить книгу»
 3. Книга появится в таблице
  ##  Как фильтровать книги
- По жанру: введите Роман в поле «Фильтр по жанру» и нажмите «Применить фильтр» — покажутся только романы
- По страницам: введите 500 в поле «Страниц больше» — останутся книги длиннее 500 страниц
- Комбинированный фильтр: укажите и жанр, и страницы одновременно
- Сбросить фильтр: нажмите «Сбросить фильтр»
## Как сохранить и загрузить данные
- Сохранить: нажмите «Сохранить в JSON» — все книги запишутся в файл books.json
- Загрузить: нажмите «Загрузить из JSON» — данные восстановятся из файла
## Как удалить книгу
- Кликните по строке с книгой в таблице (она выделится)
- Нажмите «Удалить выбранную»
